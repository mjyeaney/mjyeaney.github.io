Among most useful commands we can have at our disposal, those related to networking are handy for troubleshooting many types of problems. I'll cover some of the basic utilities here, and give a few examples of how they can be used to understand your environment. In following examples, we'll use a Windows Server 2016 VM deployed on Azure. Interfaces Get-NetAdapter ARP Get-NetNeighbor Routing Get-NetRoute DNS Resolve-DnsName Get-DnsClientCache Get-DnsClientServerAddress Connections Get-NetTcpConnection Azure Exmaples - Basic environment discovery - Can't RDP to VM - No internet access
